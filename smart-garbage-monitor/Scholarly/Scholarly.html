Descriere:

GaSM (Garbage Smart Monitor) este un site informativ cu privire asupra gunoiului colectat si reciclat din orasul Iasi, aceste cuprinde atat o pagina de statistici cat si o pagina pentru fiecare cartieri. GaSM ofera posibilitatea, unui cetatean, de a furniza comentarii asupra starii unui cartier dar si de initializa o plangere.

GaSM decongestioneaza gunoiul atat la nivel de cartier cat si pentru fiecare point selectat pe mapa, verifica fiecare plangere aplicata si selecteaza si sorteaza comentariile postate de catre utilizatori.

Manual de utilizare:

GaSM (Garbage Smart Monitor) contine in total 8 pagini web dintre care doar 7 sunt accesibile unui utilizator obisnuit, iar 1 pagina pentru contul de admin (contul salubris). Fiecare pagina contine un nav bar care contine numele paginii logo(prin care se revine la pagina principala) si un buton pentru login/pagina principala:

PAGINA PRINCIPALA:

Aceasta contine un meniu prin care se face legatura cu paginile Get Started, Learn how to do it si Statistici, dar si un buton in partea din dreapta jos al paginii ce conduce catre raportul scholarly. Pentru a accesa una din paginile dorite, atat din navbar, cat si din meniu se acceseaza printr-un singur click pe optiunea dorita.

LOGIN PAGE:

Aceasta contine un formular in care se introduce username-ul si parola contului de admin(in cazul nostru salubris), in caz de datele sunt introduse gresit sau contul este inexistent se va afisa un alert. Dupa ce se completeaza formularul se va apasa Log in pentru submit.

SALUBRIS ACCOUNT PAGE:

Aceasta pagina permite utilizatorului se adauge, stearga, verifice, modifice date din baza de date si comentarii sau reclamatii postate de catre utilizatorii neautorizati.

GET STARTED:

Pagina contine o lista cu cartierele din Iasi atat nume cat si poza pentru fiecare cartier. Click pe poza cartierului pentru a accesa pagina acestuia.

CARTIER:

Pagina contine statistici pentru cartierul respectiv cat si locul in clasament pentru cel mai curat cartier, acesta dispune de o mapa prin care se poate orienta locatia gunoiului din cartierul respectiv. Cu ajutorul formuralui din partea de jos a paginii se poate posta un comentariu sau opinie despre acel cartier.

LEARN HOW TO DO IT PAGE:

Pagina contine butoane pentru fiecare tip de gunoi, accesand unul din acestea se va afisa informatii generale despre acel tip de gunoi cat si informatii despre cum se recicleaza si de ce este bine sa se recicleze.

STATISTICS PAGE:

Pagina contine un grafic interactiv si butoane specifice pentru hartie,plastic sau organic dar si o lista de butoane pentru fiecare cartier sau luna. Pe langa butoanele mentionate pagina dispune de 3 tipuri de export: XML,CSV,PDF ce contin tabele de date sau graficul selectat.

Structura:

Tehnologiile folosite in crearea site-ului:

HTML

HTML este o formă de marcare orientată către prezentarea documentelor text pe o singura pagină, utilizând un software de redare specializat, numit agent utilizator HTML, cel mai bun exemplu de astfel de software fiind browserul web. HTML furnizează mijloacele prin care conținutul unui document poate fi adnotat cu diverse tipuri de metadate și indicații de redare. Indicațiile de redare pot varia de la decorațiuni minore ale textului, cum ar fi specificarea faptului că un anumit cuvânt trebuie subliniat sau că o imagine trebuie introdusă, până la scripturi sofisticate, hărți de imagini și formulare.

CSS

CSS (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. Stilurile se pot atașa elementelor HTML prin intermediul unor fișiere externe sau în cadrul documentului, prin elementul <style> și/sau atributul style. CSS se poate utiliza și pentru formatarea elementelor XHTML, XML și SVGL.

JavaScript

JavaScript (JS) este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor. Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul JavaScript din aceste pagini fiind rulat de către browser.

Baze de Date

PostgreSQL este un sistem de baze de date relationale. Este disponibil gratuit sub o licentă open source de tip BSD. PostgreSQL nu este controlat de nici o companie, își bazează dezvoltarea pe o comunitate răspândită la nivel global, precum și câteva companii dezvoltatoare. PostgreSQL permite folosirea limbajelor procedurale pentru a executa blocuri de cod direct în serverul de baze de date. Se pot folosi pentru a crea funcții definite de utilizator (subrutine, trigerre, agregate și funcții fereastră) sau pentru a crea blocuri ad hoc "DO".

JSPDF

JSPDF este o biblioteca open source pentru exportarea de fisiere PDF. Aplicatia WEB foloseste JSPDF pentru a exporta graficul cu statisticile totale.

PHP

PHP (acronim recursiv pentru PHP: Hypertext Preprocessor) este un limbaj de scripting de uz general, cu cod-sursă deschis (open source), utilizat pe scară largă, și care este potrivit în special pentru dezvoltarea aplicațiilor web și poate fi integrat în HTML.

Heroku

Heroku este o platforma container-based care permite developerilor sa implementeze, sa dezvolte, sa ruleze si sa opereze aplicatii in totalitate pe cloud.

Chart.JS

Chart.js este o bibliotecă JavaScript gratuită cu sursă deschisă pentru vizualizarea datelor , care acceptă 8 tipuri de diagrame : bara , linie , zonă , plăcintă ( gogoașă ), bulă , radar , polar și scatter . Creat de la Londra pe bază de web dezvoltator Nick Downie în 2013, acum este menținută de comunitate și este a doua dintre cele mai cunoscute JS bibliotecă diagrame pe GitHub de numărul de stele după D3.js , considerat în mod semnificativ mai ușor de utilizat , deși mai puținpersonalizabile decât cele din urmă. Chart.js este redat în panza HTML5 și este acoperit pe scară largă ca una dintre cele mai bune biblioteci de vizualizare a datelor. Este disponibil sub licența MIT .

Functionalitati suplimentare:

Google Maps API

The Maps JavaScript API vă permite să personalizați hărți cu conținut și imagini proprii pentru afișare pe pagini web și dispozitive mobile. The Maps JavaScript API include patru tipuri de hărți de bază (foaie de parcurs, satelit, hibrid și teren) pe care le puteți modifica utilizând straturi și stiluri, controale și evenimente și diverse servicii și biblioteci.
